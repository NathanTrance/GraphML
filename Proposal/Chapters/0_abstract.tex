\begin{abstract}
GAAP achieves state-of-the-art graph fraud detection by learning attribute-association patterns through adaptive feature binning and message passing. However, its standard cross-entropy loss treats every training node equally, causing rare fraud-indicative patterns to be drowned out by common benign ones. We propose RP-GAAP, which assigns higher sample weights to nodes with infrequent patterns, improving detection without any architectural modification. Preliminary results show Recall@K gains of $+0.97$ on T-Finance and $+0.78$ on Tolokers. Project page: \url{https://github.com/NathanTrance/GraphML}.
\end{abstract}
